トップ画面がindex.htmlです。
メインに使っていたブラウザは、GoogleChromeです。
トップ画面では、新しいTodoを作成出来ます。

todoを入力し、”リストの作成”ボタンを押すと、
下にリストが表示されます。

リストには、
タイトル、作成日、詳細、期限が表示されます。
タイトルが大きいフォントのもので、リスト作成時に入力した文字が表示されます。
作成日は、リスト作成時の日付が自動で表示されます。
リスト作成時は、詳細、期限は、なにも表示されていません。
右側には、”未完了”と表示されたボタンがあります。
そのボタンを押すと、”未完了”と表示されていたものが、”完了”に変わります。




編集ボタンを押すと、
編集ページに移動します。
編集ページは、
タイトル、詳細、期限のフォームがあります。
ページの読み込み時に、編集前のテキストがフォームに自動で表示されます。

詳細のフォームは、１６０文字まで入力出来ます。
期限のフォームをクリックすると、フォーム下にカレンダーが表示され、
カレンダーで、日付を入力できます。
この機能は、jQueryのプラグインを使っています。
完了ボタンを押すと、もとのリストのページに戻ります。

リストのページに戻ると、編集した内容が反映されています.

リセットボタンを押すと、ローカルストレージに保存したデータをリセットできます。
